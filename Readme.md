## QueueWithTimeout

## Checksum

* inn?( String )
    
  Проверка корректности ИНН

* okpo?( String )

  Проверка правильности ОКПО

* orgn?( String )

  Проверка правильности OГРН

* addr_or_name?( String )
  
  Если есть слова из букв и длина больше 10, то может быть это что-то значимое.
